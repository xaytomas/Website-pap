01 — Sobre o Projeto

O que é o
Matecnic Calendar?

O Matecnic Calendar é uma aplicação mobile desenvolvida no âmbito do Projeto de Aptidão Profissional (PAP) do curso de Gestão e Programação de Sistemas Informáticos.

O objetivo central é resolver um problema real na empresa Matecnic: a gestão de férias dos seus colaboradores. A aplicação substitui processos manuais e fragmentados por uma solução centralizada, rápida e segura.

Através de diferentes perfis de utilizador — colaborador, chefe, gerente e administrador — cada interveniente tem acesso apenas às funcionalidades relevantes para o seu cargo, garantindo uma cadeia de aprovação clara e eficiente.

02 — Funcionalidades

O que a app faz

01
Autenticação Segura

Login protegido com credenciais únicas por utilizador. Cada perfil acede apenas às funcionalidades correspondentes ao seu cargo na empresa.

02
Gestão de Utilizadores

Criação, edição e organização de todos os colaboradores da empresa com atribuição de cargo, departamento e permissões de acesso.

03
Marcação de Férias

Calendário interativo para selecionar e submeter pedidos de férias. O colaborador visualiza o estado das suas marcações em tempo real.

04
Cálculo Automático de Dias

O sistema calcula automaticamente o total de dias de férias disponíveis, usados e restantes, actualizando-se a cada aprovação ou recusa.

05
Relatórios PDF e Excel

Geração automática de documentos com o resumo das férias aprovadas — disponíveis em PDF para envio formal e Excel para análise de dados.

06
Sistema de Notificações

Notificações automáticas para cada etapa do processo: submissão, aprovação, recusa e alterações de estado das marcações.

05 — Stack Tecnológico

Linguagens &
Frameworks

Frontend · Mobile
Android Studio
Interface desenvolvida em XML + Java nativo. Layouts adaptados a cada perfil de utilizador, com ConstraintLayout e design de componentes personalizados.
Backend · API
Java + Spring Boot
API REST construída com Spring Boot. Inclui camadas de Controller, Service e Repository, com geração de PDFs, ficheiros Excel e autenticação JWT.
Base de Dados
Microsoft SQL Server 2019
Base de dados relacional para armazenamento de utilizadores, marcações, departamentos e histórico de aprovações. Gerida via SQL Server Management Studio.

07 — Análise do Projeto

Pontos fortes &
dificuldades

✓ Pontos Positivos
Praticidade

A aplicação traz uma enorme praticidade no dia a dia dos colaboradores e da empresa. O processo de marcação de férias, que antes era manual e sujeito a erros, passa a ser digital, rápido e rastreável.

✗ Pontos Negativos
Integração

A integração da aplicação com o servidor interno da empresa revelou-se a maior complexidade técnica do projeto, exigindo configurações de rede e segurança adicionais para funcionar no ambiente corporativo.

→ Oportunidades
Crescimento

Os conhecimentos adquiridos ao longo deste projeto — Spring Boot, Android nativo, SQL Server — abrem portas no mercado de trabalho. A aplicação pode ainda ser escalada para outras funcionalidades de RH.

⚠ Ameaças
Concorrência

Existem no mercado soluções comerciais que desempenham papel semelhante. A vantagem do Matecnic Calendar é ser construído especificamente para as necessidades e processos internos da empresa.
